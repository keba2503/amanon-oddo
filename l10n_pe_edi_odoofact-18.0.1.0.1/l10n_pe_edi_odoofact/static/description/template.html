¿Qué es el Tipo de Cambio SUNAT?

El tipo de cambio es la relación de equivalencia entre dos monedas. En Perú, la SUNAT publica diariamente el tipo de cambio oficial entre el Sol Peruano (PEN) y el Dólar Americano (USD), el cual es utilizado para efectos tributarios y contables.

¿Sabías que...? La SUNAT establece un tipo de cambio oficial que todas las empresas deben usar para sus declaraciones tributarias. Con este módulo, estarás siempre al día.

Instalación y configuración

Requisito previo: Token de conexión

Para utilizar este módulo es obligatorio contar con un Token para la API. Sin el token, no se podrá actualizar el tipo de cambio automáticamente.

https://www.operu.pe/r/comprar-token-ilimitado

Documentación completa disponible

Consulta la guía paso a paso con capturas y ejemplos prácticos:

https://www.operu.pe/r/manual_facturacion_electronica

Recomendamos revisar la documentación antes de iniciar la configuración

Pasos de instalación

  1. Instala el módulo desde la tienda o ZIP.
  2. Ve a Ajustes → Usuarios y compañías → Compañías.
  3. En la compañía, ingresa el TOKEN y activa la validación por token.
  4. Haz clic en Guardar.

Nota: Una vez configurado, el sistema actualiza el tipo de cambio a las 9:00 AM (hora de Perú) cada día.

Comprobación y uso

Para verificar la actualización:

  1. Contabilidad → Configuración → Monedas
  2. Selecciona USD
  3. Abre la pestaña Tasas
  4. Revisa el historial actualizado

Configuración manual (opcional)

Para ingresar una tasa manual en una fecha específica:

  1. Contabilidad → Configuración → Monedas
  2. Selecciona USD
  3. Pestaña Tasas → Crear
  4. Ingresa fecha y tipo de cambio → Guardar
