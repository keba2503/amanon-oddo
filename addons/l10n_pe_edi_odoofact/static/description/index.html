¿Qué es OdooFact?

OdooFact es un módulo de facturación electrónica para Perú que permite la emisión de comprobantes electrónicos según la normativa de SUNAT. Se integra específicamente con PSE/OSE Nubefact para el envío y validación de documentos electrónicos ante SUNAT, permitiendo una gestión completa de facturas, boletas, notas de crédito, notas de débito y otros documentos fiscales electrónicos.

Importante: Este módulo requiere una cuenta activa con Nubefact para funcionar correctamente.

Características principales

Comprobantes Electrónicos

Emisión de facturas, boletas, notas de crédito y notas de débito electrónicas que se envían a SUNAT a través de la plataforma Nubefact.

Integración con Nubefact

Conexión directa con la API de Nubefact (PSE/OSE) para el envío automático de comprobantes a SUNAT sin intervención manual.

Envío Automático al Cliente

Envío automático de comprobantes por correo electrónico, incluyendo PDF y XML formato SUNAT, cumpliendo la normativa vigente.

Configuración Completa

Configuración de series, numeración, catálogos SUNAT y tipos de afectación de IGV según normativa peruana.

¿Cómo funciona?

Flujo simple: Odoo → Nubefact → SUNAT

1. Configura

Configura credenciales de Nubefact y parámetros de tu empresa en Odoo.

2. Emite

Emite tus facturas en Odoo; el módulo las envía automáticamente a Nubefact.

3. Valida

Nubefact procesa y envía a SUNAT, retornando el estado de validación.

Resumen de Instalación

  1. Instala el módulo l10n_pe_edi_odoofact desde las aplicaciones de Odoo.
  2. Ve a Contabilidad → Configuración → Ajustes.
  3. En Localización Peruana, busca Facturación Electrónica.
  4. Activa la opción de facturación electrónica.
  5. Configura tus credenciales de Nubefact (Token y RUC).
  6. Configura las series de tus comprobantes.
  7. Haz clic en Guardar.

Importante: Este es un resumen; sigue el manual completo enlazado.

Configuración de Credenciales

Para conectar con Nubefact necesitarás:

  • Token de Nubefact (panel de Nubefact).
  • RUC de la empresa registrado en Nubefact.
  • Modo de prueba o producción.
  • Series (F001, B001, etc.).
  • Datos de empresa (razón social, dirección, etc.).

Importante: Asegúrate de que tu cuenta de Nubefact esté activa y configurada.

¿Qué es Nubefact?

Nubefact es un Proveedor y Operador de Servicios Electrónicos (PSE/OSE) autorizado por SUNAT para el envío de comprobantes electrónicos.

  • Autorizado por SUNAT como PSE/OSE.
  • API robusta y confiable.
  • Soporte técnico especializado.
  • Planes accesibles para todo tipo de empresas.
